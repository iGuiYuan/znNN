The znNN.jar program was used to prediction of zinc-binding sites of proteins.
The details of running the program can be found in the user manual.
PSFM_profile.pl is used to generate PSFM profile in the manuscript. There is a simplied PSFM in 
the PSSM profile and the simplied PSFM was directly used in this release version.
